## Practical synthesis for this project
- CC can be biased when missingness is informative.
- IPW helps but can be unstable / less efficient.
- Doubly robust estimators can be protective when one nuisance model is approximately correct.
- Strong MNAR remains challenging for MAR-based approaches.
